# 🛋️ موقع أبو الخير لشراء الأثاث المستعمل بالمنصورة

موقع تعريفي احترافي (صفحة هبوط) تم بناؤه باستخدام **HTML5** و **CSS3** ومخصص لخدمات تجارة الأثاث والعفش المستعمل في مدينة المنصورة وضواحيها. الموقع مصمم ليكون خفيفاً وسريعاً جداً ومتوافقاً بالكامل مع شاشات الهواتف الذكية لضمان تجربة مستخدم ممتازة وتصدر نتائج بحث جوجل (SEO).

## 🚀 المميزات الفنية للموقع
* **بدون تعقيدات برمجية:** يعتمد على كود كلاسيكي نظيف يضمن استقرار الموقع بنسبة 100% على منصة Vercel.
* **سرعة فائقة:** يتم تحميل الصفحة في أجزاء من الثانية لعدم وجود ملفات JavaScript ثقيلة.
* **متوافق مع الهواتف:** التصميم مرن (Responsive) ويتكيف تلقائياً مع كافة أحجام الشاشات.
* **زر اتصال فوري:** يحتوي على زر اتصال مباشر يسهل على العملاء التواصل عبر الهاتف بنقرة واحدة.

## 📁 هيكل المشروع
المشروع يتكون من بنية بسيطة ومباشرة:
* `index.html`: الملف الرئيسي الذي يحتوي على الأكواد، النصوص العربية، والتصميم (CSS).
* `README.md`: هذا الملف التوثيقي للمشروع.

## 🌐 النشر والاستضافة
الموقع مستضاف ومرفوع مجاناً على منصة **Vercel السحابية**، ويتم تحديثه تلقائياً فور تعديل أي نص داخل ملف الـ HTML على GitHub.
